Methods

static authVia(platform, page)

A wrapper for the 3rd party authentication service. Handles fetching of redirect URL, all redirect activities, and storing of session.

Parameters:
Name Type Description
platform string

The platform to authenticate onauthentication is happening on - this is a workaround to only having one callback URL.
page string

The name of the page in which the page is redirected FROM. NOTE: The redirect URL after success/failed auth is set on the BACKEND, it cannot be controlled from the client.
